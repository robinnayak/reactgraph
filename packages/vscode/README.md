# ReactGraph

Visualize your React/Next.js project as an interactive
hierarchy graph inside VS Code.

## What it does

Maps your entire codebase visually:
**Pages -> Components -> Hooks -> APIs**

- See which components are used where
- View props and TypeScript types on each node
- Detect which pages break when you change a file
- Run a Health Check before building for production

## How to use

1. Open any React or Next.js project in VS Code
2. Press `Ctrl+Shift+P`
3. Type `ReactGraph: Open Graph`
4. Press Enter

## Features

- Interactive hierarchy graph with zoom and pan
- Props and TypeScript types shown on every node
- Click any node to inspect dependencies
- Impact Analysis - see what breaks if you change a node
- Health Check - find TypeScript errors before production build
- Shared component detection across pages
- Works with Next.js App Router and Pages Router

## Requirements

- A React or Next.js project with TypeScript
- VS Code 1.80 or higher
